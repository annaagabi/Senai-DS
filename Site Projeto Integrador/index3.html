1º Semestre


DISCIPLINA LINGUAGEM DE MARCAÇÃO

Carga Horária: 75 horas

Professor: Miguel Alberto

Dias da Semana: Toda sexta-feira

Objetivos: Proporcionar a aquisição de capacidades básicasrelativas ao leiaute, codificação e estilização de páginas web por meio de linguagem de marcação, bem como o desenvolvimento de capacidades socioemocionais adequadasa diferentes situações profissionais.

Unidade Curricular:

1. Linguagens de marcação

2. World wide web (www)

3. Leiautes de tela

4. HTML

5. Estilização de páginas

6. Imagens (UX) design

Atividades Solicitadas Pelo Professor:

1. Criação de Curriculo

2. Lista Colorida

3. Prototipagem no Figma

4. Atividade HTML e CSS

5. Criação De Site Para Restaurante
