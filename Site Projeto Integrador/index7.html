3º Semestre


Disciplina Programação Para Dispositicos Móveis

Carga Horária: 100 horas

Objetivos: Proporcionar a aquisição de capacidades técnicas relativas ao desenvolvimento de aplicativos para dispositivos móveis e sua integração com demais plataformas, bem como o desenvolvimento de capacidades socioemocionais adequadas a diferentes situações profissionais.

Unidade Curricular:

1. APIs

2. Persistência de dados em dispositivos móveis

3. Consumo de RESTfull web service

4. Publicação do aplicativo

Disciplina Requisitos e Modelagem de Software

Carga Horária: 45 horas

Objetivos: Proporcionar a aquisição de capacidades técnicas relativas ao levantamento, análise e proposição de soluções para atender as necessidades do cliente, bem como o desenvolvimento de capacidades socioemocionais adequadas a diferentes situações profissionais.

Unidade Curricular:

1. Requisitos

2. Levantamento de Requisitos

3. Gerenciamento de Requisitos

4. UML (Unified Modeling Language)

5. Documentação de Requisitos

Disciplina Testes de Software

Carga Horária: 30 horas

Objetivos: Proporcionar a aquisição de capacidades técnicas relativas a elaboração de roteiro de teste e execução de cenários de testes que atenda às necessidades do cliente, bem como o desenvolvimento de capacidades socioemocionais adequadas a diferentes situações profissionais.

Unidade Curricular:

1. Testes

2. Planejamento de testes

3. Execução de testes

Disciplina Projetos

Carga Horária: 225 horas

Objetivos: Proporcionar a aquisição de capacidades técnicas relativas ao planejamento, desenvolvimento e implantação de sistemas computacionais com foco nas necessidades do cliente, bem como o desenvolvimento de capacidades socioemocionais adequadas a diferentes situações profissionais.

Unidade Curricular:

1. Qualidade de software

2. Metodologias de desenvolvimento

3. Metodologia de gerenciamento de projeto

4. Apresentação do projeto
